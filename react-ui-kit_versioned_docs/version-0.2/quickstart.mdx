# Quickstart

Here is how you can integrate Dyte's UI Kit into your project seamlessly.

import { loadMeeting } from '/src/utils/meeting';
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
  groupId="ui-quick"
  defaultValue="code"
  values={[
    { label: 'Code', value: 'code' },
    { label: 'Preview', value: 'preview' },
  ]
}>
<TabItem value="code">

```jsx
import { useDyteClient } from '@dytesdk/react-web-core';
import { DyteMeeting } from '@dytesdk/react-ui-kit';

function App() {
  const [meeting, initClient] = useDyteClient();

  useEffect(() => {
    initClient({
        authToken: '<auth-token>',
        roomName: '<room-name>',
      });
  },[])

  return (<DyteMeeting meeting={meeting} />);
}
```

This will render our user interface **covering the entire width and height of the page** .

If you want it to only fill the parent dimensions use `mode="fill"`

```jsx
<div style={{height:'50vh', width: '50vw'}}>
  <DyteMeeting meeting={client} mode="fill" />
</div>
```

:::caution Height

It is important for the parent component to have well defined height when using `mode="fill"`

:::
