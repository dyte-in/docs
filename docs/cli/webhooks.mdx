# Webhooks

Instead of polling our APIs, you can use webhooks to get information as it is produced.

## Creating a webhook
Run the following command to create a webhook.
```
dyte webhooks create
? What is the name of your webhook? My Webhook 1
? Enter the URI to be used for delivering the webhook: https://webhook.example.com
? Enter the events you want to subscribe to (use "space" to select/unselect): meeting.started, meeti
ng.ended
? Do you want to activate this webhook? Yes
```
